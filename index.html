Welcome!

GURPS Character Sheet (GCS) is a stand-alone, interactive, character sheet editor that allows you to build characters for the GURPS 4th Edition roleplaying game system. This site is the home for GCS, which was written by Richard A. Wilkes.

Version 4.24


Released September 18, 2020

GCS v4.23 (and newer) can no longer open the old file formats from GCS v4.19.1 and earlier. Before upgrading to a current release of GCS, please make sure you've loaded and re-saved your custom library files and character sheets with GCS v4.22.1 (really, any version from v4.20 up to v4.22.1 will work, but might as well use the last release with support for the old format to ensure the best conversion). You can also do this in bulk via the command line with the --loadsave flag.

Please join us on the GURPS Discord for discussion about GCS — specifically the #apps and #apps-dev channels. You can find me there as Rich (GCS).

Trying to learn how to use GCS? A short blog article on The Basics of Using GCS might help. Give the author some encouragement and he may produce more.
